# 🚀 Дорожная карта развития проекта Family Finance

## 📊 Текущее состояние

### ✅ Реализовано
- Базовый учёт транзакций (доходы/расходы)
- Категории транзакций
- Разделение между пользователями
- Расчёт долгов между пользователями
- ML-аналитика (прогнозы, аномалии, здоровье)
- Автокатегоризация
- Финансовое здоровье
- Демо-режим

### ⚠️ Частично реализовано (есть в БД, но не в UI)

- Прикрепление чеков (поле есть, UI нет)
